# Creating First Simple Application

## Introduction

Flask is a lightweight Python web framework used to create web applications and APIs. It is simple, flexible, and beginner-friendly, making it a popular choice for learning web development.

In this tutorial, you will learn how to:

- Install Flask
- Create your first Flask application
- Run a Flask server
- Handle GET and POST requests
- Use HTML forms with Flask templates

---

# Prerequisites

Before starting, make sure you have:

- Python 3.x installed
- pip (Python package manager)
- Basic knowledge of Python

---

# Step 1: Install Flask

Open your terminal or command prompt and run:

```bash
pip install Flask
```

This command installs Flask and its required libraries.

---

# Step 2: Create Your First Flask Application

Create a file named `app.py`.

Add the following code:

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello World'

if __name__ == '__main__':
    app.run()
```

---

# Code Explanation

## Create Flask Application

```python
app = Flask(__name__)
```

This creates the Flask application object.

---

## Define a Route

```python
@app.route('/')
```

The route `/` represents the homepage URL.

---

## Create a Function

```python
def hello_world():
    return 'Hello World'
```

When users visit the homepage, Flask calls this function and displays:

```text
Hello World
```

---

## Run the Server

```python
app.run()
```

Starts the local Flask development server.

---

# Step 3: Run the Flask Application

Run the application using:

```bash
python app.py
```

You will see output similar to:

```text
 * Running on http://127.0.0.1:5000/
```

Open the URL in your browser:

```text
http://127.0.0.1:5000/
```

Output:

```text
Hello World
```

---

# Step 4: Handling POST Request Using HTML Form

Now let’s create a form that sends data to Flask using a POST request.

---

# Project Folder Structure

```text
project/
│
├── app.py
│
└── templates/
    └── name.html
```

---

# Create HTML Form

Inside the `templates` folder, create a file named `name.html`.

```html
<!DOCTYPE html>
<html>
  <body>
    <h3>Enter Your Name</h3>

    <form method="post">
      <input type="text" name="username" required />
      <br /><br />

      <input type="submit" value="Submit" />
    </form>
  </body>
</html>
```

---

# HTML Form Explanation

## Form Method

```html
<form method="post"></form>
```

Uses the POST method to send data to the server.

---

## Input Field

```html
<input type="text" name="username" />
```

The entered value is sent using the key `username`.

---

## Submit Button

```html
<input type="submit" value="Submit" />
```

Sends the form data to Flask.

---

# Update app.py

Replace the previous code with:

```python
from flask import Flask, request, render_template

app = Flask(__name__)

@app.route('/login', methods=['GET', 'POST'])
def login():

    if request.method == 'POST':
        name = request.form['username']
        return f"Hello {name}, POST request received"

    return render_template('name.html')

if __name__ == '__main__':
    app.run(debug=True)
```

---

# Flask POST Request Explanation

## Import Required Modules

```python
from flask import Flask, request, render_template
```

- `request` handles incoming form data
- `render_template()` loads HTML files

---

## Allow GET and POST Methods

```python
@app.route('/login', methods=['GET', 'POST'])
```

- `GET` displays the form
- `POST` processes submitted data

---

## Read Form Data

```python
name = request.form['username']
```

Reads the value entered in the form input.

---

## Display HTML Template

```python
return render_template('name.html')
```

Loads the HTML file from the `templates` folder.

---

## Enable Debug Mode

```python
app.run(debug=True)
```

Debug mode automatically reloads changes and shows detailed errors during development.

---

# Run the Application

Start the Flask server:

```bash
python app.py
```

Visit:

```text
http://127.0.0.1:5000/login
```

Enter your name and click **Submit**.

Example output:

```text
Hello John, POST request received
```
